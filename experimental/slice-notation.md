# [Slice notation][proposal-slice-notation]

## SliceExpression

```ts
interface SliceExpression <: ChainElement, Expression {
    type: "SliceExpression";
    object: Expression;
    lower: Expression | null;
    upper: Expression | null;
}
```

A slice expression, e.g. `arr[0:1]`.
