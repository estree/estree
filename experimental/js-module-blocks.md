# [JS Module Blocks][proposal-js-module-blocks]

## Expressions

### ModuleExpression

```js
interface ModuleExpression <: Expression {
    type: "ModuleExpression";
    body: Program;
}
```

The `sourceType` of the `body` must be `"module"`.
