This document specifies the extensions to the core ESTree AST types to support the ES2019 grammar.

# Statements

## CatchClause

```js
interface CatchClause <: Node {
    type: "CatchClause";
    param: Pattern | null;
    body: BlockStatement;
}
```

The `param` is `null` if the `catch` binding is omitted. E.g., `try { foo() } catch { bar() }`
